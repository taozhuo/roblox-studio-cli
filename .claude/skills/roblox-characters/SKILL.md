# Characters Reference

## Quick Reference

- **Adaptive Animation**: Set up your character rig to support universal sets of animations and emotes.
- **Character appearance**: Customize your in-experience character appearance and properties.
- **Emotes**: Emotes are character animations used through chat commands, or accessing the emotes menu.
- **Characters**: Characters are interactive models that interact with the world or other users.
- **Character name/health display**: You can customize character UI, like name and health displays, using Class.Humanoid.
- **Character pathfinding**: Pathfinding is the process of moving a character along a logical path to reach a destination.
- **R6 to R15 Adapter**: The R6 to R15 adapter enables R15 avatar characters to join your R6 experience.
- **Photo-to-Avatar generation**: Explains how to use AvatarCreationService AvatarGeneration APIs to prompt users to generate Avatars from photos in experience.
- **Avatar in-experience creation**: Explains how to use AvatarCreationService to prompt users to create and purchase avatars from within your experience.
- **Create avatar items**: Create and upload avatar characters, clothing, and accessories to the Roblox Marketplace.
